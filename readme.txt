Allows logging into you Wordpress site using the verifyne.me app.

== Description ==
This is the Wordpress plugin for use with the verifyne.me android app. It allows you to login to you Wordpress site without having to enter a username or password while increasing security.

All you have to do is scan a QR code, confirm the login on your android smartphone and you are logged in automatically.

Verifyne uses modern elliptic curve (ed25519) cryptography for highest security. It allows user authentication without having to trust a third-party or foreign servers.

Get the most advanced password-less login solution today:

*   NO need to remember a password or even a username to login to your wordpress site
*   NO third-party trust relationshit
*   Increase security with public key cryptography
*   Keys are stored and encrypted on your smartphone and optionally also protected with a password
*   Try and experience the smoothest login ever

Note that if you want to get the maximum out of the plugin you should install the sodium library libsodium on your host.

The plug-in is available on [GitHub](https://github.com/sruester/wp-verifyne).
Visit [verifyne.me](https://verifyne.me) for more information.

== Installation ==
Install the plugin, go to your profile page, scroll to the bottom and connect your profile with a verifyne identity.
